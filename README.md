# Hexa_Decimus
# Application Tracker

A cross-platform mobile application that helps users organize and manage their job search. Users can track job applications, upload resumes and cover letters, schedule follow-up reminders, and monitor their application progress through an interactive dashboard.

## Tech Stack

- **Frontend:** Flutter
- **Backend:** Java
- **Database:** MySQL
- **Version Control:** Git & GitHub

## Features

- User authentication (Register/Login)
- Job application management
- Document Vault for resumes and cover letters
- Automated follow-up email reminders
- Application pipeline dashboard

## Git Workflow

- Do **not** commit directly to `main`.
- Create a new feature branch for each task.
- Submit a Pull Request before merging.
- Resolve merge conflicts before requesting a review.
