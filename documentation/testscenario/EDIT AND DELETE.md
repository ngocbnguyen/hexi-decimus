# Test Scenario: Editing and Deleting Applications

# Editing Applications Details

With an testing application ready, I ran the PUT(edit) request with the proper parameters and sent it. The local host returned the changes. Next came the history check to see the changes side by side. With a POST request to the history log, I was able to get the most recently changed application and the application before the change.
